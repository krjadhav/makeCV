## Current Scope

- What: Build a web-based collaborative LaTeX editor where users can create and edit documents in real time, compile to PDF, and preview output directly in the browser. Team collaboration should be link-based for fast sharing and access. The editor should include LaTeX syntax highlighting and error visibility.
- Core Value: Instant and clear compilation feedback.
- In Scope:
  - Real-time collaborative LaTeX editing
  - In-browser LaTeX PDF preview
  - Preview blocked when syntax/compilation errors exist
  - Syntax highlighting with error visibility in editor
  - Link-based sharing for team members
  - Export/download document as PDF
- Out of Scope:
  - Subscription and billing systems (defer monetization)
  - Additional integrations (keep v1 focused)
  - Project management features (avoid scope creep)
  - Rich commenting and review workflows (defer advanced collaboration)
- Constraints:
  - No hard stack or timeline constraints defined
  - Must work seamlessly on web
